#### ver 0.2 업데이트 사항
- 기존 구조체로 구현되어 있던 account 자료형을 클래스로 구현
- 클래스 내부 메소드 및 생성자 소멸자 구현 완료
- 입출금 잔액 입력 시 금액이 0보다 작거나 같을 때에 대한 예외처리 완료
- account 클래스 내부에 name 변수를 입력받은 이름의 길이에 따라 동적할당하도록 변경
